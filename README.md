# MERN-SHOP

## Description

Mern-Shop is a full stack e-commerce website built using MERN Stack + TypeScript

## Stack I Used

-   FrontEnd: React, React-Router, Redux + Redux Toolkit, bootstrap + react-bootstrap ...
-   Backend: Express, mongoose, jwt, bcrypt ...
